📋 Cómo obtener el link de Google Sheets:
1. Abra su hoja en Google Sheets
2. Archivo → Publicar en la web
3. Seleccione la hoja y formato CSV
4. Haga clic en Publicar y copie el link
✓ El link termina en ...output=csv
☁️ Cómo obtener el link de OneDrive:
1. Abra el archivo en OneDrive
2. Haga clic en Compartir → Cualquiera con el vínculo
3. Copie el link y péguelo aquí
4. La app lo convierte a descarga directa
✓ Formatos: .xlsx, .xls, .csv
🏢 Cómo obtener el link de SharePoint:
1. Abra el archivo en SharePoint
2. Haga clic en los 3 puntos (...) → Compartir
3. En permisos elija "Cualquiera con el vínculo"
4. Copie el link y péguelo aquí

⚙️ Para descarga directa desde SharePoint:
Cambie :x:/g/ por :x:/r/ y agregue ?download=1 al final
⚠️ Requiere que el archivo sea público o anónimo
